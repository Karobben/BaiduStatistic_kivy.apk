# BaiduStatistic_kivy

基于kivy的百度统计的 .安卓 显示自己主页的各项数据.  开源, 有需要, 自己改.
apk 可直接在 `bin` 文件夹中找到.
复制好自己的token后, 点击左上角的"搜索词", 即可黏贴入token, 祝好.

仅适用普通账户
